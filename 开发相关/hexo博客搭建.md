| Hexo常用命令  | 作用                                           |
| ------------- | ---------------------------------------------- |
| hexo s(erver) | 启动本地服务器                                 |
| hexo new "AA" | 新建"AA"文章                                   |
| hexo d        | 部署到设定的仓库                               |
| hexo clean    | 清除缓存文件 db.json 和已生成的静态文件 public |
| hexo g        | 生成网站静态文件                               |
